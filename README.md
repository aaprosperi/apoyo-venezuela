# Apoyo Venezuela

Una plataforma comunitaria para conectar y apoyar a venezolanos en momentos de incertidumbre.

## Características

- **Muro de Apoyo**: Comparte mensajes de esperanza y solidaridad
- **Ofrecer Ayuda**: Publica ofertas de alojamiento, trabajo, alimentos, etc.
- **Buscar Ayuda**: Encuentra asistencia cuando la necesites
- **100% Gratuito**: Sin costo alguno, hecho con amor para la comunidad

## Tecnologías

- HTML5, CSS3, JavaScript vanilla
- LocalStorage para persistencia de datos
- Diseño responsivo
